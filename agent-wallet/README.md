# agent-wallet

Scoped, revocable session signing for autonomous agents.

Issue a session key with bounded capabilities — which chain, which contracts, max spend, expiry — so an agent can act on your behalf without ever holding your master key. Revoke any time.

## Chains

Arc · Giwa · BNB
