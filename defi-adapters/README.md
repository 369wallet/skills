# defi-adapters

Plug-in DeFi protocol adapters.

A registry and base interface (`describe`, `quote`, `execute`) so new protocols on Arc, Giwa, and BNB can be added without bespoke wiring. Agents discover and use them at runtime.

## Chains

Arc · Giwa · BNB
